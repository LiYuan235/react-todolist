# react-todolist

用react简单实现的todolist，能够简单实现统计todolist事件的数量，添加事件等

这个例子也是我在学react中的简单小案例，希望能给react初学者一点帮助

# 使用方法

直接clone到本地之后，直接运行html文件即可
